# goalManagement
> 目標管理アプリ

<p>目標を管理したい自分のためのアプリ。</p>

## 基本機能
<ul>
<li>目標を設定・編集・削除することができる</li>
<li>毎日のtodoを表示し、管理することができる</li>
<li>過去の達成率を確認することができる</li>
<li>ご褒美の設定ができる</li>
</ul>
